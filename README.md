# krom_node_api

A contact book REST API built with:
- Node.JS
- `Express` server
- `MongoDB` via `mongoose`
- `crypto` library

The API is able to:
- Create a contact
- Read a contact
- List all contacts
- Update a contact
- Delete a contact

The crypto npm package is used to:
- Encrypt the data before it's stored
- Decrypt the data when performing `READ` and `UPDATE` operations
